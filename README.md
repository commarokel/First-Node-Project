# First-Node-Project

Personal Project to replicate StackOverflow's features. 

## Main Features

1. Account creation with Passport JS
2. Question Submission
3. Answer Submission
4. Commenting System 
5. Ajax refresh of Answers & Comments
6. Voting System for Questions & Answers

#### List of Packages

1. Express Routing
2. Express Flash
3. Passport JS
4. Sequelize 
5. Express Validator
6. Express Session
